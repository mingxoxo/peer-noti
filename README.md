## Introduction
**peer noti**는 42Seoul 동아리 [42peer](https://www.peer-study.co.kr/)의 알림 서비스입니다.

## Key Features

- 42 Slack workspace에 있는 유저에게 슬랙봇을 이용한 DM 전송
- 메시지 전송용 관리자 페이지
